Sobre o Projeto

Visualize um robô animado em realidade aumentada. Este modelo 3D possui múltiplas animações que podem ser controladas através dos botões na interface. Use o marcador Hiro para ativar a experiência AR e explore as diferentes animações disponíveis.

Iniciar Experiência AR

Como usar:

  1. Clique no botão "Iniciar Experiência AR" acima
  2. Permita o acesso à câmera quando solicitado
  3. Imprima ou exiba o marcador Hiro em uma tela
  4. Aponte a câmera para o marcador
  5. O modelo 3D do robô aparecerá sobre o marcador
  6. Use os botões ⏮ e ⏭ para navegar entre as animações
