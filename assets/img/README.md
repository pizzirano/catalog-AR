# Pasta de Imagens

Coloque aqui as imagens dos projetos do catálogo.

## Arquivos necessários:

- **phoenix.jpg** - Imagem da Fênix (recomendado: 400x300px ou proporção similar)
- **robot.jpg** - Imagem do Robô (recomendado: 400x300px ou proporção similar)  
- **shenron.jpg** - Imagem do Shenron (recomendado: 400x300px ou proporção similar)

## Formatos suportados:
- JPG/JPEG
- PNG
- WebP

## Tamanho recomendado:
- Largura: 400-800px
- Altura: 300-600px
- Proporção: 4:3 ou 16:9

As imagens serão exibidas automaticamente no catálogo após serem adicionadas nesta pasta.
